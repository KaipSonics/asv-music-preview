# ASV Production — генератор референсов 🎧

Веб-инструмент для музыкального сервиса [ASV Production](http://www.asvproduction.ru).
Артист собирает **вайб будущего трека** по элементам (бит / бас / мелодия),
выбирает настроение и темп, а нейросеть генерирует короткий **инструментальный
референс**. Понравившийся вариант артист отправляет в студию по коду заявки.

🔗 **Живое демо:** https://asv-music-preview-production.up.railway.app

> Запросы к нейросети идут с сервера (вне РФ), поэтому пользователю **VPN не нужен**.

## Как это работает

1. Выбор **жанра по элементам**: бит / бас / мелодия (POP, POP DANCE, HOUSE,
   HIP-HOP, TRAP, DNB, ROCK, DANCEHALL) + **настроение** + **темп** (Slow/Mid/Fast).
2. Сервер собирает **единый образ трека** (бит = главный жанр ~50%, бас и мелодия —
   вторичные влияния, настроение и темп — на весь трек) и отправляет в GenAPI.
3. Нейросеть (**Suno** через GenAPI) генерирует инструментальный референс.
4. Сохраняются последние **3 референса** — артист сравнивает и выбирает лучший.
5. Артист копирует **код заявки** (`ASV-XXXXXX`) и идёт в [VK ASV](https://vk.com/asv_family).
   По коду студия находит нужный референс в истории GenAPI.

## Особенности

- Прослушивание в плеере (**превью ≤ 60 секунд**, скачивание не предусмотрено).
- **Лимит 3 генерации в сутки** с одного IP.
- Код заявки и комментарий артиста зашиты в название генерации — студия находит
  референс в истории GenAPI по коду.
- История последней сессии сохраняется в `localStorage`.

## Стек

- **Next.js 15** (App Router) + **React 19** + **TypeScript**
- Серверный API: `app/api/generate` (старт) + `app/api/status` (опрос — генерация
  Suno асинхронная, ~2.5 мин)
- Генерация: **GenAPI** (Suno). Провайдер переключается переменной `AUDIO_PROVIDER`:
  `genapi` · `demo` (демо-клипы) · `fal` · `replicate` · `huggingface`
- Хостинг: **Railway** (деплой `railway up`)
- Дизайн: тёмный космос, неон `#62e1d8`, шрифты **Dune Rise** (логотип) +
  **Syncopate** + Montserrat / Inter

## Переменные окружения

| Переменная        | Назначение                                              |
| ----------------- | ------------------------------------------------------- |
| `AUDIO_PROVIDER`  | `genapi` (боевой) или `demo` (демо-клипы без генерации) |
| `GENAPI_KEY`      | Ключ GenAPI (gen-api.ru)                                |
| `GENAPI_MODEL`    | Модель GenAPI (по умолчанию `suno`)                     |

Полный список — в [`.env.example`](.env.example). Реальные ключи задаются в Railway → Variables.

## Локальный запуск

```bash
npm install
cp .env.example .env.local   # впиши GENAPI_KEY, либо AUDIO_PROVIDER=demo
npm run dev                  # http://localhost:3000
```

## Деплой

Подключён к Railway. Деплой из папки проекта:

```bash
railway up
```
